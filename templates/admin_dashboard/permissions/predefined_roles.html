About Predefined Roles

Predefined roles are pre-configured sets of permissions designed for common user responsibilities in the system. These roles can be applied to users to quickly grant them appropriate permissions for their job functions.

Benefits of using predefined roles:

  • Consistent permission assignment across users with similar responsibilities
  • Simplified user management
  • Reduced risk of permission errors
  • Easy onboarding of new users

To apply a predefined role to a user, go to the User Management page, edit the user, and assign them to the appropriate group.
